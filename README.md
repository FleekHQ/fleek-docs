## Getting started

### Prerequisites

- python >= 3.7.4
- pip >= 20.1.1

### Commands

Before executing any mkdocs command, make sure to install [mkdocs-material](https://github.com/squidfunk/mkdocs-material) by running:

```sh
pip install mkdocs-material
```

#### Run Locally

```sh
mkdocs serve
```

#### Build

```sh
mkdocs build
```
